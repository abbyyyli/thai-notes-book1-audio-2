Thai Speaking 1｜Day 14 學習指南

主題：家庭成員、父母、子女、排行、兄弟姊妹、祖父母、外祖父母、孫子孫女、年齡與年齡比較

Textbook-based Supplement / Local usage Review Common mistake

Lesson Overview

Textbook-based Day 14 的核心是「家庭與年齡」。教材先整理家庭成員，例如父母、兒子、女兒、老大、老二、老么、獨生子女、兄弟姊妹、哥哥/弟弟/姊姊/妹妹、孫子/孫女、祖父母與外祖父母。接著練習「你有幾個兄弟姊妹」、「你家有幾個人」、「你幾歲」、「誰年紀比較大/比較小」、「差幾歲」、「年紀一樣」。

Age comparison
พี่สาวอายุมากกว่าฉันห้าปีค่ะ
phîi-sǎaw aa-yú mâak gwàa chǎn hâa bpii khâ
My older sister is five years older than me.
Review： Day 13 已學比較句 กว่า / gwàa / than。Day 14 把它放到年齡裡：อายุมากกว่า / aa-yú mâak gwàa / older than、อายุน้อยกว่า / aa-yú nɔ́ɔi gwàa / younger than。
Bangkok Practical Focus： 這課很適合用在泰國朋友、老師、男友家人、語言交換、初次聊天。泰國人聊天常問家庭、兄弟姊妹、排行與年齡，但如果不熟，年齡問題可以用比較柔和的問法。

本課學習路線

  1. 能介紹家庭有幾個人、自己排行與兄弟姊妹。
  2. 能說年齡、誰比較大或小。
  3. 能避免把泰文的哥哥/姊姊/弟弟/妹妹混用。

Supplement｜本課延伸整理

家庭稱謂邏輯｜泰文先看年紀，再看性別

核心：中文說「哥哥、弟弟、姊姊、妹妹」也看年齡；泰文更常把這個資訊直接放進稱呼裡。先判斷對方比你大還是小，再選 พี่ 或 น้อง。

5. Older than / younger than / same age

Textbook-based
A + อายุมากกว่า + B + number + ปี
A + aa-yú mâak gwàa + B + number + bpii
A is number years older than B.
A + อายุน้อยกว่า + B + number + ปี
A + aa-yú nɔ́ɔi gwàa + B + number + bpii
A is number years younger than B.
A + กับ + B + อายุเท่ากัน
A + gàp + B + aa-yú thâo gan
A and B are the same age.
CaseSentenceChinese Explanation
A 30, B 25
A อายุมากกว่า B ห้าปีค่ะ
A aa-yú mâak gwàa B hâa bpii khâ
A is five years older than B.
 มากกว่า = more than，用在年齡就是比較大。
B 25, A 30
B อายุน้อยกว่า A ห้าปีค่ะ
B aa-yú nɔ́ɔi gwàa A hâa bpii khâ
B is five years younger than A.
 น้อยกว่า = less than，用在年齡就是比較小。
C 20, D 20
C กับ D อายุเท่ากันค่ะ
C gàp D aa-yú thâo gan khâ
C and D are the same age.
 เท่ากัน = equal / the same。

Textbook-based
A + กับ + B + ใครอายุมากกว่ากันคะ

A + กับ + B + ใครอายุน้อยกว่ากันคะ

本課最重要記憶點：

1. 人的量詞是 คน / khon / person：พี่น้องสามคน / phîi-nɔ́ɔng sǎam khon / three siblings。

3. 父系祖父母：ปู่ ย่า / bpùu, yâa / paternal grandparents；母系祖父母：ตา ยาย / dtaa, yaai / maternal grandparents。
