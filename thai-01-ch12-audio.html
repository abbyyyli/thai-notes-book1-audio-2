Thai Speaking 1｜Day 12 學習指南

主題：曾經做過、做過幾次、第一次/最後一次、早中晚時間段、昨晚、幾點鐘、出發/抵達、開門/關門

Textbook-based Supplement / Local usage Review Common mistake

Lesson Overview

Textbook-based Day 12 的核心是兩條線：第一條是「經驗」，例如你曾經去過哪裡、吃過什麼、做過幾次、第一次或最後一次是什麼時候。第二條是「時間」，包含早上、上午、中午、下午、傍晚、晚上、深夜，以及泰式幾點鐘說法。

Review： Day 11 已經學過「多久、星期、日期、生日」。Day 12 會把時間變得更口語、更真實：你會開始聽到 กี่โมง / gìi moong / what time、กี่ทุ่ม / gìi thûm / what hour at night、ตอนเช้า / dtɔɔn cháao / morning。
Bangkok Practical Focus： 這課最實用的場景是：問朋友去過哪裡、問餐廳幾點開/關、問昨晚幾點吃飯、問 Grab 或朋友幾點出發/到達、說自己第一次去某地、最後一次去某店是什麼時候。

本課學習路線

  1. 能問是否去過、做過幾次。
  2. 能用第一次、最後一次說經驗。
  3. 能問商店開幾點、什麼時候出發或到。

Sentence Structures

1. Have you ever...?

Textbook-based
คุณเคย + V. + Object + ไหมคะ
khun khəəy + V. + Object + mái khá
Have you ever + verb + object?
QuestionPositiveNegative
คุณเคยไปเชียงใหม่ไหมคะ
khun khəəy bpai Chiang Mai mái khá
Have you ever been to Chiang Mai?
เคยไปแล้วค่ะ
khəəy bpai lɛ́ɛw khâ
I have been there already.
ไม่เคยไปค่ะ
mâi khəəy bpai khâ
I have never been.

2. How many times have you...?

Textbook-based
คุณเคย + V. + Object + กี่ครั้งแล้วคะ
khun khəəy + V. + Object + gìi khráng lɛ́ɛw khá
How many times have you done it already?
QuestionAnswer TypeExample
คุณเคยกินทุเรียนกี่ครั้งแล้วคะ
khun khəəy gin thú-rian gìi khráng lɛ́ɛw khá

 Exact number
เคยกินสองครั้งแล้วค่ะ
khəəy gin sɔ̌ɔng khráng lɛ́ɛw khâ
I have eaten it twice already.
คุณเคยไปตลาดนี้กี่ครั้งแล้วคะ
khun khəəy bpai tà-làat níi gìi khráng lɛ́ɛw khá
How many times have you been to this market?

เคยไปหลายครั้งแล้วค่ะ
khəəy bpai lǎai khráng lɛ́ɛw khâ
I have been many times already.
คุณเคยไปภูเก็ตกี่ครั้งแล้วคะ
khun khəəy bpai Phuu-gèt gìi khráng lɛ́ɛw khá
How many times have you been to Phuket?

เคยไปหนึ่งครั้งเท่านั้นค่ะ
khəəy bpai nʉ̀ng khráng thâo-nán khâ

3. First time / nth time / last time + when?

Textbook-based
คุณ + V. + Object + ครั้งแรก / ครั้งที่... / ครั้งสุดท้าย + เมื่อไรคะ
khun + V. + Object + khráng rɛ̂ɛk / khráng thîi... / khráng sùt-tháai + mʉ̂a-rài khá
When did you do it for the first / nth / last time?
QuestionAnswer
คุณไปเชียงใหม่ครั้งแรกเมื่อไรคะ
khun bpai Chiang Mai khráng rɛ̂ɛk mʉ̂a-rài khá
When did you go to Chiang Mai for the first time?
ไปครั้งแรกเมื่อสองปีก่อนค่ะ
bpai khráng rɛ̂ɛk mʉ̂a sɔ̌ɔng bpii gɔ̀ɔn khâ
I went for the first time two years ago.
คุณไปครั้งสุดท้ายเมื่อไรคะ
